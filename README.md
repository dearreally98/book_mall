# 📚 Book Mall

HTML, CSS, JavaScript를 활용하여 제작한 **도서 쇼핑몰 웹 사이트**입니다.

사용자는 도서를 탐색하고, 상세 정보를 확인하며, 장바구니에 담고, 회원 정보를 수정하는 등 쇼핑몰의 기본적인 기능을 경험할 수 있습니다.

---

# 📖 프로젝트 소개

Book Mall은 웹 프론트엔드 기술(HTML5, CSS3, JavaScript)을 활용하여 제작한 팀 프로젝트입니다.

도서 쇼핑몰을 주제로 하여 사용자 친화적인 UI를 구현하고 JavaScript를 이용한 동적인 기능을 추가하여 실제 쇼핑몰과 유사한 서비스를 구현하는 것을 목표로 하였습니다.
